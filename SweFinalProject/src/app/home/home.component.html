Please Read

Please be patient when viewing each page, as it takes some time to load the data from each API call. You can get more visual feedback on what is happening by viewing the outputs in the developer console.


List Issues

This page allows you to view the top issues for the Repo Group you selected. This data is derived from a custom API endpoint.

Repo Group 20 is a good example for viewing data.


Compare Commits

This page compares the two Repos you chose by displaying their total number of commits in a bar graph.

"accumulo-examples" and "actionmailbox" are good examples for viewing data.


Compare Issues

This page compares the two Repos you chose by displaying their total number of reported issues in a bar graph.

"accumulo-examples" and "actionmailbox" are good examples for viewing data.
